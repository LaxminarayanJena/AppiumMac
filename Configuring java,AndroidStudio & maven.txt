Configuring environment variable for java
------------------------------------------------------------
Open terminal
1)touch.bash-profile
2)open .bash-profile
3)export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk1.8.0.jdk/Contents/Home
4)export PATH=$PATH:$JAVA_HOME/bin
5)close bash profile
6)open bashprofile
echo $ JAVA-HOME
